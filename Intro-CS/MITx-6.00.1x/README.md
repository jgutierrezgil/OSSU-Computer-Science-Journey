## 🛠️ Configuración del Entorno

### Requisitos
- Python 3.7 o superior
- PyCharm EDU o VSCode
- Git para control de versiones

### Instalación
```bash
# Crear entorno virtual
python -m venv venv

# Activar entorno virtual

pip install -r requirements.txt
```
